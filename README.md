# LED-pythonApp

## Setup

Clone the repository
```
git clone https://github.com/Hiebeler/LED-pythonApp.git
```

Install required packages
```
pip install -r requirements.txt
```

Run the project (with root privileges)
```
sudo python3 LEDmain.py
```
